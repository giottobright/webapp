## 🎬 Анимации

### 1. Hover на карточке
```
Исходное состояние:
┌──────────┐  ↑ 0px
│  [ФОТО]  │
│ Elif, 23 │
└──────────┘

При наведении:
┌──────────┐  ↑ 4px (поднимается)
│ [ФОТО+5%]│      (фото увеличено на 5%)
│ Elif, 23 │
└──────────┘
   + тень становится больше
```

### 2. Открытие детального экрана
```
Шаг 1: Фон затемняется
░░░░░░░░░  (0% прозрачности)
▒▒▒▒▒▒▒▒▒  (50%)
▓▓▓▓▓▓▓▓▓  (95% черного)

Шаг 2: Модалка выезжает снизу
       ┌─────┐
       │     │
       │     │
       └─────┘ ← Снизу вверх за 0.4 секунды
```

### 3. Пульсация статуса
```
● ← 100% яркость
○ ← 50% яркость
● ← 100% яркость

Повторяется каждые 2 секунды
```

### 4. Нажатие на кнопку
```
Обычное состояние:
┌────────────────┐
│  Кнопка (100%) │
└────────────────┘

При нажатии:
┌──────────────┐
│ Кнопка (95%) │  ← Уменьшается до 95%
└──────────────┘
  + вибрация (haptic feedback)
```

## 🔊 Тактильная обратная связь

```
Лёгкое касание (light):
  • При клике на карточку
  • При закрытии модалки
  ≈ Лёгкая вибрация

Среднее касание (medium):
  • При нажатии на кнопку выбора
  ≈ Средняя вибрация

Уведомление (success):
  • При успешном выборе девушки
  ≈ Короткая мелодичная вибрация
```

## ✨ Особенности дизайна

### Что делает его премиум:

1. **Черный фон** вместо серого
   - Современно, элегантно, премиум

2. **Градиенты везде**
   - Заголовки, кнопки, статусы
   - Розовый → Фиолетовый → Синий

3. **Большие фото**
   - 3:4 соотношение (как портрет)
   - Заполняют всю карточку

4. **Плавные анимации**
   - Cubic-bezier для естественности
   - 60fps для плавности

5. **Тактильная обратная связь**
   - Вибрация как в нативных приложениях

6. **Детальная информация**
   - Не просто имя, а полный профиль

7. **Статус онлайн**
   - С пульсацией для живости

8. **Стильные теги**
   - В виде пилюль с эффектами
